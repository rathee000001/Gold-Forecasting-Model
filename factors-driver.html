Factors & Drivers

This page shows the factor inputs used to explain monthly gold returns (USD). In the workbook, the Factors_Model sheet converts raw series into returns and standardized (Z-score) variables used by the regression model.

What you are seeing:

  • Gold_Return: actual monthly gold return (target series).
  • Driver returns / levels: USD, rates/inflation (real-rate logic), import proxy, geopolitical risk, and reserve share change.
  • Standardization (Z): makes different scales comparable before estimating coefficients.

Model equation (Z-score form):

Gold_Return_Z = β₀
+ β₁ · US_Import_Return_Z
+ β₂ · USD_Return_Z
+ β₃ · Real_Rate_Z
+ β₄ · GPR_Index_Z
+ β₅ · CB_BRICS_Share_Change_Z

Accuracy Metrics & Interpretation

Metrics below are computed from the workbook (Factors_Model sheet) using Gold_Return (Actual) vs Predicted_Return (Forecast).

MAD
avg(|A − F|)
0.0252
MSE
avg((A − F)²)
0.00109
MAPE
avg(|(A − F)/A|) × 100%
129.27%

How to interpret these (returns-based):

  • MAD: average absolute monthly return error is ~0.0252 (≈ 2.52 percentage points if returns are in decimals).
  • MSE: penalizes large misses (squared error), useful to detect shock-period blow-ups.
  • MAPE: very large here because monthly returns can be near zero (division makes % errors explode). For returns, MAD/MSE are usually more stable than MAPE.
